# 세일즈 어시스턴트 기획서 v2

## 1. 프로젝트 재정의

### 1.1 프로젝트명
**세일즈 어시스턴트 (Sales Assistant)**
AI 기반 세일즈 심리 전략 도구

### 1.2 핵심 컨셉
> "단순 응대 도구가 아니라, 세일즈 심리 전략 도구"

- 고객에게 **어떤 인식**을 남길 것인가
- 고객에게 **어떤 감정**을 불러일으켜야 하는가
- AI가 고객의 **현재 감정을 캐치**해서 알려줌
- 이 상황에서 **어떤 멘트**를 해야 목표 인식을 강화/극대화할 수 있는가

### 1.3 목적
구글맵 SEO 상품 판매 시:
- 고객 심리를 실시간으로 파악
- 목표하는 인식/감정으로 유도하는 전략적 응답 생성
- 일관된 세일즈 퍼포먼스 유지
- 세일즈 기획과 실행의 갭 최소화

### 1.4 앱의 역할

```
Notion = 데이터 저장소 (백오피스)
  - 고객 CRM 관리
  - 스크립트 편집/관리
  - 자료 정리
  - 세일즈 전략 문서

앱 = 상담 중 사용하는 AI 코파일럿 (프론트)
  - 고객 정보 빠르게 확인
  - 고객 감정 분석 (핵심!)
  - 전략적 응답 생성 (핵심!)
  - 원클릭 복사
  - 스크린샷 분석
```

## 2. 세일즈 심리 전략 프레임워크

### 2.1 목표 설정

#### 목표 인식 (Perception Goals)
고객이 나에 대해 가져야 할 인식:
- "이 사람은 전문가다"
- "이 사람은 믿을 수 있다"
- "이 사람은 내 상황을 이해한다"
- "이 서비스는 효과가 있다"
- "가격 대비 가치가 있다"

#### 목표 감정 (Emotional Goals)
고객이 느껴야 할 감정:
- 신뢰감
- 안심
- 기대감
- 긴급함 (놓치면 안 될 것 같은)
- 확신

#### 페르소나 설정
내가 보여줄 이미지:
- 전문적이면서 친근한
- 밀어붙이지 않지만 확신에 찬
- 고객 입장에서 생각하는

### 2.2 고객 감정 분석

#### 감정 카테고리
```
긍정적 신호:
- 관심 (Interest): "더 알고 싶어요"
- 기대 (Expectation): "효과 있을까요?"
- 신뢰 (Trust): "믿어볼게요"
- 결정 (Decision): "해볼게요"

중립적 신호:
- 탐색 (Exploring): "여러 군데 알아보고 있어요"
- 정보 수집 (Gathering): "자료 좀 주세요"

부정적 신호:
- 의심 (Doubt): "정말요?", "다른 데도..."
- 저항 (Resistance): "비싸요", "지금은..."
- 무관심 (Indifference): 짧은 답변, 느린 응답
- 이탈 (Leaving): "생각해볼게요", "나중에..."
```

### 2.3 전략적 응답 생성

#### AI 분석 프로세스
```
1. 입력
   - 고객 정보 (Notion CRM)
   - 대화 내역 (카톡 스크린샷 → OCR)
   - 세일즈 전략 (목표 인식/감정)

2. 분석
   - 고객 현재 감정 캐치
   - 현재 상담 단계 파악
   - 목표와의 갭 분석

3. 출력
   - "지금 고객은 [감정] 상태입니다"
   - "[목표 감정]으로 전환하려면..."
   - "이렇게 말하세요: [멘트]"
   - 관련 스크립트 추천
```

#### 상황별 전략

| 고객 감정 | 목표 전환 | 전략 |
|-----------|-----------|------|
| 의심 → | 신뢰 | 구체적 사례, 숫자 제시 |
| 저항(가격) → | 가치 인식 | ROI 관점, 비교 프레임 |
| 탐색 → | 관심 | 차별점 강조, 호기심 유발 |
| 망설임 → | 결정 | 긴급함, 리스크 제거 |
| 무관심 → | 관심 | 질문으로 니즈 환기 |

- 카톡 대화 내용 (텍스트 붙여넣기)

  - "첫 상담이에요"
  - "가격 얘기 나왔어요"
  - "거의 결정 단계예요"

│                                         │

│ 🎯 목표 감정: 확신 → 결정               │
